# 🔒 Password Strength Analyzer

A Python-based desktop application that analyzes password strength using common cybersecurity best practices. The application evaluates passwords based on length, uppercase letters, lowercase letters, numbers, and special characters, then provides a strength rating along with suggestions to improve weak passwords.

---

## 📌 Features

- Analyze password strength
- Detect uppercase and lowercase letters
- Check for numeric characters
- Check for special characters
- Password strength rating (Weak, Medium, Strong)
- Personalized improvement suggestions
- Show/Hide password option
- Clear input functionality
- User-friendly Tkinter GUI

## 🔍 Password Evaluation Criteria

The application checks whether the password contains:

- At least 8 characters
- Uppercase letters (A–Z)
- Lowercase letters (a–z)
- Numbers (0–9)
- Special characters (e.g. ! @ # $ % ^ & *)

Based on these criteria, the password is classified as:

- 🔴 Weak
- 🟠 Medium
- 🟢 Strong
